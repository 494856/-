如何使用
建议：最好按 ID 添加，因为如果按用户名添加，则只会添加具有用户名的用户。而且，它很慢

安装要求
pip install -r requirements.txt

阅读完整教程并使用它HOW_TO_USE.txt

添加您的帐户以添加和抓取成员

python manager.py

刮擦成员
python scraper.py

添加成员
python tsadder.py

首先，您需要在 manager.py 中存储用于抓取和添加目的的帐户
所以在终端启动

> python manager.py

--要添加新帐户，请选择 1
然后输入您要添加的帐户数量
然后输入你的电话号码
之后，您必须从新添加的帐户登录
输入登录代码和 2FA 密码 [如果您已设置] 并继续

--要过滤掉所有被禁止的帐户，请选择选项2

--要删除特定帐户，请选择选项3
选择要删除的任何帐户并继续

现在我们需要将成员添加到组中

所以在终端启动

> 蟒蛇添加.py

输入您想从中刮取用户的群组链接
它可以是私人链接或公共链接

公共grp链接的一般格式

https://t.me/用户名

私人群组链接的一般格式

https://t.me/join

***** 笔记 *****

如果你以前从一个组刮过，Astra 会记住它，它会问你是否刮
是否来自同一组。输入 y 表示是，输入 n 表示否并继续
******************

然后输入您要使用的帐户数

=每个帐户将尝试添加 60 个成员=

然后选择是添加到公共组还是私人组

然后输入组链接并继续

然后您需要输入每个请求的延迟时间

它是什么？

这是 Astra 在添加下一个用户之前将等待的时间

输出是
*********
将一个人添加到组
--------
等待几秒钟
--------
将另一个人添加到同一组


==如果你想快速添加，那么你可以设置延迟时间为0，这意味着Astra在添加用户时不会等待
但是从 Telegram 获得对等洪水错误的机会更多==


建议的延迟是

5 秒、10 秒、15 秒、20 秒或 60 秒，如果你愿意的话，但它太慢了 XD

然后，坐下来观看 Astra 添加成员
